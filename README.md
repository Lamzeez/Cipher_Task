**"CipherTask" - Secure Encrypted To-Do System**

Mobile Secure Data Storage & Management with Secure Communication and Networking (MVVM) in Flutter 

====================================================================================================

This project successfully implements:
- AES-256 Encryption for data at rest.
- Secure local databases using SQLCipher (via Hive with encryption).
- Biometric Authentication and Automatic Session Timeouts.
- Utilized Hardware-Backed Key Storage (Android Keystore) via flutter_secure_storage.
- Strict adherence to the MVVM Architectural Pattern.
